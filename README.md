# 🎻 Enzira Begena LMS

**Enzira Begena** is a premium, high-performance Learning Management System (LMS) designed for academic excellence. Built with a focus on minimalist aesthetics and surgical precision, it provides a seamless experience for both students and administrators.

## ✨ Key Features

### 👨‍🎓 Student Experience
- **Fluid Navigation**: Animated sidebar and floating mobile bottom-nav for effortless discovery.
- **Modern Course Player**: Optimized learning interface with resource management.
- **Personalized Dashboard**: Real-time progress tracking and course recommendations.
- **Adaptive UI**: Fully responsive design that feels native on mobile, tablet, and desktop.

### 🛡️ Admin Command Center
- **Master-Detail Submissions**: Efficient workflow for verifying manual payment receipts.
- **Course Builder**: Modular curriculum creation with multimedia support.
- **Institutional Controls**: Global system settings, maintenance mode, and security configurations.
- **Real-time Analytics**: High-impact dashboards tracking revenue and enrollment metrics.

---

## 🎨 Design Philosophy

Enzira Begena follows a strict **Modern SaaS Aesthetic**:
- **Unified Geometry**: A consistent `5px` border-radius baseline for all UI elements.
- **Minimalist Palette**: High-contrast, clean layouts using curated HSL color tokens.
- **Micro-Animations**: Smooth width/opacity transitions and hover effects to improve engagement.
- **Glassmorphism**: Subtle backdrop-blur effects on navigation and overlays.

## 🚀 Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/)
- **Components**: Custom UI Library (Shadcn-inspired)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🛠️ Getting Started

### Prerequisites
- Node.js 18.x or later
- pnpm (recommended) or npm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/natnaelkornima/Enzira_begena_LMS.git
   cd Enzira_begena_LMS
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
